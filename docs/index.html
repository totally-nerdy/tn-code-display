The web component to display code - examples

The component use's highlight.js with style monokai-sublime as a default. Following languages are supported as of now: CSS, SCSS, Less, JavaScript and HTML. Don't hesitate to add the languages you need and create a PR. We're happy about every contribution.

Please find a real-world example in our blog post "How to add a "Go to Top" button to your Rocketspark website".
